# Product Explorer & Error-Resilient Logger

This project is part of Assignment Part 3.

## Features:
- File handling (read and write)
- API data fetching using DummyJSON
- Filtering and sorting products
- Exception handling
- Input validation
- Error logging system

## Files:
- part3_api_files.py → main code file
- python_notes.txt → generated file
- error_log.txt → stores error logs

## How to Run:
Run the Python file:
python part3_api_files.py
